# 🤡 Joker Mind AI

> "Let's put a smile on that terminal!"

## 🎭 What is Joker Mind AI?

Joker Mind AI is a **terminal-based AI assistant** specially designed for **Kali Linux & Parrot OS**. It helps hackers, pentesters, and security enthusiasts by:

- ✅ Explaining any command in **Hindi** (type in English!)
- ✅ Fixing errors with smart suggestions
- ✅ Having Joker-style conversations
- ✅ Working offline & online (with Gemini AI)

---

## 🚀 Features

| Feature | Description |
|---------|-------------|
| 🎯 **Command Explainer** | 100+ Kali commands with Hindi meanings |
| 🔧 **Error Fixer** | Detects errors & suggests fixes |
| 💬 **Chat Mode** | Talk freely like ChatGPT |
| 🤡 **Joker Persona** | Funny, dark humor responses |
| 📡 **Hybrid Mode** | Local DB + AI (Gemini) |
| ⚡ **Fast & Lightweight** | No heavy dependencies |
