## 🎭 What is Joker Mind AI?

**Joker Mind AI** is a terminal-based AI assistant specially designed for **Kali Linux** and **Parrot OS**. It helps ethical hackers, pentesters, and cybersecurity enthusiasts by:

✅ **Explaining ANY Kali command in Hindi** (you type in English!)  
✅ **Fixing errors** with smart, actionable suggestions  
✅ **Chatting freely** like ChatGPT but inside your terminal  
✅ **Joker-style dark humor** to keep things interesting  
✅ **Working offline** (local database) + **online** (Gemini AI)
